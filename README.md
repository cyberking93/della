# Della Quick Order — Android

Android-приложение для платформы [Della](https://mob.della.kz) — биржи грузов и перевозок.
Главное назначение — **быстрое размещение заказа на груз** в один тап.

## Что умеет

- Открывает мобильный сайт `https://mob.della.kz` внутри нативного WebView, сохраняя авторизацию (cookies).
- **Кнопка «Быстрый заказ»** (FAB и пункт меню) — мгновенный переход на форму создания груза `/app/my/freights/create`.
- Меню навигации: Главная, Мои грузы, Поиск грузов, Обновить, Открыть в браузере.
- Pull-to-refresh, индикатор загрузки, кнопка «Назад» (история WebView), обработка SSL/сетевых ошибок.
- Загрузка файлов (фото/документы) через системный picker — для добавления фото груза.
- Системные **App Shortcuts** (долгое нажатие на иконку): «Быстрый заказ», «Поиск грузов».
- Локализация: русский (по умолчанию), казахский, английский.

## Сборка

Требования:
- Android Studio Hedgehog/Iguana или новее
- JDK 17
- Android SDK 34 (compile/target), minSdk 24 (Android 7.0)

### Через Android Studio
1. `File → Open` → выбрать корень проекта.
2. Дождаться синхронизации Gradle.
3. Запустить на устройстве/эмуляторе (`Shift+F10`).

### Из командной строки
```bash
# первая сборка — Gradle сам докачает wrapper'ы:
./gradlew :app:assembleDebug

# APK будет тут:
ls app/build/outputs/apk/debug/app-debug.apk
```

Если в проекте нет `gradlew`-скриптов, сгенерируйте их из Android Studio (`File → Sync` создаст wrapper) или вручную: `gradle wrapper --gradle-version 8.6`.

## Использование

1. Запустите приложение — откроется `https://mob.della.kz`.
2. Войдите в аккаунт Della (cookies сохранятся между запусками).
3. Нажмите большую кнопку **«Быстрый заказ»** внизу — откроется форма создания груза.
4. Заполните: описание, пункты загрузки/выгрузки, даты, параметры — и отправьте.
5. Долгое нажатие на иконку приложения → ярлык «Быстрый заказ» сразу открывает форму, минуя главный экран.

## Замечания

- Это WebView-обёртка вокруг мобильного сайта Della. Все бизнес-правила (валидация, тарифы, верификация) выполняются на стороне сервера della.kz.
- Камера и галерея требуются для прикрепления фото груза. Разрешения запрашиваются стандартным системным пикером.
- Для нативного API-клиента понадобится официальная документация API Della (на данный момент отсутствует публично).
